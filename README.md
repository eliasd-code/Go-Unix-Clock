Debian based:\n
sudo apt update golang

Arch based:\n
sudo pacman -S golang

go get -e github.com/inancgumus/screen
go build main.go
./main

Simply run: 
go get -e github.com/inancgumus/screen
go run main.go


Run Compiled:
chmod +x x64_main
./x64_main
